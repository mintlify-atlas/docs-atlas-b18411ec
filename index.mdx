<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your CryptoPulse API running in minutes</p>
  
  <Steps>
    <Step title="Set up environment variables">
      Copy the example environment file and configure your credentials:
      
      ```bash
      cp .env.example .env
      ```
      
      Update `ADMIN_USER`, `ADMIN_PASS`, `JWT_SECRET`, `DATABASE_URL`, `REDIS_URL`, and `COINGECKO_API_KEY` in your `.env` file.
    </Step>
    
    <Step title="Start with Docker Compose">
      Launch PostgreSQL, Redis, and the API service:
      
      ```bash
      docker compose up --build
      ```
      
      The API will be available at `http://localhost:3000` and Swagger docs at `http://localhost:3000/docs`.
    </Step>
    
    <Step title="Authenticate and fetch prices">
      First, get your access token:
      
      ```bash cURL
      curl -X POST http://localhost:3000/auth/login \
        -H "Content-Type: application/json" \
        -d '{"username":"admin","password":"secret"}'
      ```
      
      <Accordion title="Example Response">
      ```json
      {
        "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "tokenType": "Bearer",
        "expiresIn": "1h"
      }
      ```
      </Accordion>
      
      Then fetch cryptocurrency prices with your token:
      
      ```bash cURL
      curl http://localhost:3000/v1/price/bitcoin \
        -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
      ```
    </Step>
  </Steps>
</div>
